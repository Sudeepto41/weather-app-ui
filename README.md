# weather-app-ui

## Pre - Requisites
1. Nodejs
2. npm

## Set up workspace
1. Run `git clone https://github.com/Sudeepto41/weather-app-ui.git` in git bash
2. cd into cloned directory

## Set up workspace
3. Open terminal in current directory
4. Run `cd weather-app-ui`.
5. Run `npm i`.

## Start dev server
6. Run `ng serve`
